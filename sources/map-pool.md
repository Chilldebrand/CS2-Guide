# CS2 map pool — checked July 13, 2026

## Premier / Active Duty

Ancient, Cache, Dust II, Inferno, Mirage, Nuke, and Anubis.

## Competitive-only

Train, Vertigo, Overpass, Office, Italy, Boulder, Fachwerk, and Shelter.

Active Duty maps are playable in Competitive as well. Premier uses the Active Duty pool. Competitive-only includes both defusal and hostage maps, so not every map uses bomb-site terminology.

## Sources

- [Valve — Season 5, Armory, and More](https://store.steampowered.com/news/posts/?appids=730) — July 8, 2026. Confirms Cache replaced Overpass in Active Duty and Boulder, Fachwerk, and Shelter entered Competitive.
- [ProSettings — All CS2 Maps](https://prosettings.net/blog/cs2-maps/) — updated July 9, 2026. Provides the current mode table and wider Competitive list.
- [Valve — The Return of Cache](https://www.counter-strike.net/newsentry/701021228894257190) — April 28, 2026. Confirms Cache entered Competitive before its Active Duty promotion.

## Caveat

This is a dated snapshot, not a permanent promise. If Valve rotates the pool, update this file first, then review map folders for obsolete callouts or utility.
