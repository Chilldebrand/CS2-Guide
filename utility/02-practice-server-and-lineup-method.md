# Practice server and lineup method

Use a private practice server to learn purpose before speed. A typical local setup is:

```text
map de_mirage
sv_cheats 1
mp_roundtime 60
mp_roundtime_defuse 60
sv_grenade_trajectory_prac_pipreview true
sv_grenade_trajectory_prac_trailtime 8
ammo_grenade_limit_total 5
mp_buy_anywhere 1
mp_buytime 9999
```

These commands can change. If one is rejected, use the current practice commands shown by the game or workshop map.

For every lineup:

1. Know the target and the teammate action it enables.
2. Repeat it five times from the same starting reference.
3. Check the landing point, not just whether the grenade “looked right.”
4. Practice the missed-throw fallback.
5. Re-check after a map update.

Learn five useful grenades on a map before adding a sixth.
