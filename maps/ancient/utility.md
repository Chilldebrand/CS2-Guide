# Ancient — utility priorities

| Target | Timing | Thrower | Purpose | Follow-up / fallback |
|---|---|---|---|---|
| T smoke Top Mid | T start | T lurker | Blocks the first CT sightline so the Mid pair can take space | Fall back and hold the smoke edge |
| T smoke Donut from Mid | Mid control | Support | Isolates A defenders while A Main players scale | Use a flash through A Main instead |
| T smoke Temple from A Main | A execute | Support | Creates a safe plant lane | Plant default and save one smoke for CT |
| T molly Cave on B hit | Before entry | Entry support | Forces the close B player out of the first duel | Clear the box with a flash |
| CT smoke B Doors on retake | After plant | Rotator | Buys a safe route into site | Save it for the tap if the first retake stalls |

## Visual lineups

### T smoke Top Mid

**Side / role:** T / T lurker<br>
**Timing:** T start<br>
**Purpose:** Blocks the first CT sightline so the Mid pair can take space

1. **Start:** Set against the right side of the large pillar in T Spawn, around the middle of the structure, as shown by the remote guide. [Remote source - not stored locally](https://gameriv.com/counter-strike-2-essential-smoke-lineups-for-ancient/)
2. **Aim:** Aim at the top of the wooden gate, with the crosshair between the second and third stone pyramid shapes, then jump-throw. [Remote source - not stored locally](https://gameriv.com/counter-strike-2-essential-smoke-lineups-for-ancient/)
3. **Result:** The smoke blocks the Red Room sightline so the Mid pair can take space without giving the first CT a clean view. [Remote source - not stored locally](https://gameriv.com/counter-strike-2-essential-smoke-lineups-for-ancient/)

**Fallback:** Fall back and hold the smoke edge
**Source:** [GameRiv - Counter-Strike 2 Essential Smoke Lineups for Ancient](https://gameriv.com/counter-strike-2-essential-smoke-lineups-for-ancient/) (checked 2026-07-13; referenced section: “Mid Smoke”)

### T smoke Donut from Mid

**Side / role:** T / Support<br>
**Timing:** Mid control<br>
**Purpose:** Isolates A defenders while A Main players scale

1. **Start:** Line up against the stone window frame outside the Mid entrance from T base. [Remote source - not stored locally](https://gameriv.com/counter-strike-2-essential-smoke-lineups-for-ancient/)
2. **Aim:** Aim at the top of the binding rope beside the scaffold and use a stationary left-click throw. [Remote source - not stored locally](https://gameriv.com/counter-strike-2-essential-smoke-lineups-for-ancient/)
3. **Result:** The smoke lands in Donut, isolating that A-side route while the A Main players scale into the site. [Remote source - not stored locally](https://gameriv.com/counter-strike-2-essential-smoke-lineups-for-ancient/)

**Fallback:** Use a flash through A Main instead
**Source:** [GameRiv - Counter-Strike 2 Essential Smoke Lineups for Ancient](https://gameriv.com/counter-strike-2-essential-smoke-lineups-for-ancient/) (checked 2026-07-13; referenced section: “Mid-Donut Smoke”)

### T smoke Temple from A Main

**Side / role:** T / Support<br>
**Timing:** A execute<br>
**Purpose:** Creates a safe plant lane

1. **Start:** Use the shared A-execute lineup position near A Main demonstrated by the remote guide. [Remote source - not stored locally](https://gamenoter.com/blog/CS2_Ancient_A_Site_Entry_Utility_Guide)
2. **Aim:** Follow the guide's Temple video for the crosshair reference and throw input; the page does not publish a reliable pixel landmark in text, so verify it in practice before use. [Remote source - not stored locally](https://gamenoter.com/blog/CS2_Ancient_A_Site_Entry_Utility_Guide)
3. **Result:** The Temple smoke blocks the deep site anchor so the entry pair can clear the near site and plant in the protected lane. [Remote source - not stored locally](https://gamenoter.com/blog/CS2_Ancient_A_Site_Entry_Utility_Guide)

**Fallback:** Plant default and save one smoke for CT
**Source:** [GameNoter - CS2 Ancient A Site Entry Utility Guide](https://gamenoter.com/blog/CS2_Ancient_A_Site_Entry_Utility_Guide) (checked 2026-07-13; referenced section: “Triple Smoke Lineup: Temple, CT Lane, and Donut”)
