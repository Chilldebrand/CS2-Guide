1. Starting roles: attackers split into an Exterior Kennels/Dock access pair, a Yard/Breezeway access pair, and a flexible escort; defenders isolate Playroom and Pool while keeping one player able to contest the return route.
2. Information trigger: confirmed Playroom or Pool access starts the rescue decision only after Courtyard, Breezeway, or Cat Adoptions information proves a safe exit.
3. Rescue/trade path: the diagram keeps the Playroom/Dock and Pool/Yard room routes distinct, then sends the forward rifle through the checked corridor while the second escort protects the carrier to the rescue zone.

## Win condition

Attackers need a safe path to and from the hostage; defenders win by breaking the escort with circular routes.

## Learn first

1. Learn the hostage-room callouts and the two safest approaches.
2. Keep the rescue carrier protected by a close escort.
3. Practice the room-entry flash and return-route smoke.
4. Review one failed rescue decision after the match.
