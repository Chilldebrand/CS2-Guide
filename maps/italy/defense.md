# Italy — defender plan

1. Keep one defender near the hostage and one near the return lane; do not overstack the first alley.
2. Use close holds only when a teammate can trade.
3. When attackers take upper space, delay the return route rather than chasing through smoke.
4. Call whether the carrier is moving or still inside.
5. Against an eco, keep distance in alleys and clear balcony corners.

## Round rule

The rescue route is the fight. Delay, split the escort, and focus the carrier instead of chasing a distant player.
