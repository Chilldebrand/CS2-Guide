# Train utility visual assets

Lineup asset names use `<map>-<side>-<grenade>-<target>-<stage>`, where `<stage>` is `start`, `aim`, or `result`.

Each lineup card must identify the exact target from `maps/train/utility.md`; copy its purpose and timing exactly; document start, aim, throw type, result, teammate action, fallback, source, checked date, and map version.

A source image may remain remote when reuse permission or geometry cannot be verified. Label it `Remote source - not stored locally` and link the source page.
