# CS2 New Player Competitive Guide Design

This repository implements the approved folder-based Markdown guide for new CS2 Competitive/Premier players. It covers 15 5v5 maps as of July 13, 2026, with separate map folders and offense, defense, utility, and visual-source files.

The guide prioritizes ten essential general principles, repeatable rank-up behavior, simple map playbooks, and utility with a stated purpose. Hostage maps use attacker/defender objective language. Map overview images are attributed remote references in the first pass; future annotated local drawings can be added under each map's `assets/` folder.
